Exam Instructions

Guidelines and instructions:
  1. This online examination is being conducted for Qualexe user practice.
  2. This online quiz tool is fully computerized and user friendly with security features & transparent evaluation system.
  3. Qalexe users can attempt the quiz with their smartphones / Desktop / Laptop / Tablet with good internet connection.
A. General information for Quiz:
  1. The quiz will comprise of Objective type Multiple Choice Questions (MCQs).
  2. Quiz has a total number of 50 questions.
  3. Each question carries 1 marks.
  4. Each correct answer has 1 marks.
  5. No negative marks for each wrong answer.
  6. The total number of MCQs, time duration and total marks details will be displayed during examination.
B. Information & Instructions:
  1. Each student will get questions in random order for the same quiz.
  2. Each MCQ has four options, User has to click on the radio button to choose the answer.
  3. Users must have to login to apply for the quiz.
  4. The Timer will start once the user clicks on the start button.
  5. Question answer navigation:
    1. User has to click on the radio button to give the answer to the question.
    2. Once the user clicks on the radio button and clicks on the next button, the answer will be saved for that question.
    3. Users can change the answer by clicking the respective radio button during the exam.
    4. Click on the clear button to clear the response.
    5. Once the timer is over, the quiz will be submitted automatically.
    6. In case of interruption in internet connectivity during the exam, the user response will be saved automatically till system restore and the timer will be adjusted with the server accordingly.
    7. The question palettes displayed on the right side of screen will show the status of each of the question using one of the following symbols:

      Answered

      Answered and Marked for review

      Not Answered and Marked for review

      Not Answered

      Not Attempted

    8. Users can move to any question by clicking on the question number from the question navigation panel.
    9. Users can finish the quiz by the submit button.
    10. After submitting and confirming the response, users will not be allowed to attempt the quiz again.
    11. User has to attempt a quiz on or before the deadline.
After Exam:
  1. User will see his/her result and detailed statistics in his/her profile.
  2. Detailed solution of only test series available till deadline of exam
  3. Detailed solution of the tournament available till first week of next month of that particular tournament round.
